Compilador/Interpretadores -> Converter o código fonte em linguagem de máquina.


* Compilador [Características]:

1. Enquanto o código fonte não estiver completamente sem erros sintáticos, a gente não consegue se quer ver a cara do programa.

2. Arquivo executável gerado não tem portabilidade para outras plataformas de sistema operacional.

3. Arquivo executável gerado pode ser executado em outros computadores independente de ter o compilador instalado.


* Interpretador

1. O código fonte não precisa estar 100% sem erros de sintaxe para executado.

2. Portabilidade: O mesmo código fonte pode ser executado em qualquer computador que tenha qualquer sistema operacional, desde que o o programa interpretador esteja instalado.

3. Para executar um programa criado utilizando uma linguagem interpretada precisa ter o interpretador instalado.
