 Pergunta 1
2 / 2 pts
Qual parâmetro de configuração IP fornece o endereço IP de um dispositivo de rede que um computador usaria para acessar a Internet?
  gateway padrão
  

Refer to curriculum topic: 2.1.2
Os tipos de configuração IP em um dispositivo de host incluem:

    Endereço IP – identifica o host na rede.
    Máscara de sub-rede - identifica a rede em que o host está conectado.
    Gateway padrão – identifica o dispositivo de redes que o host usa para acessar a Internet ou outra rede remota.
    Servidor DNS - identifica o servidor usado para traduzir um nome de domínio em um endereço IP.
